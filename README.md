# Tarot
Tarot prediction web application.

## How it works?

On the frontend user chooses prediction type and enters prompt. When user clicks submit button, backend receives request, chooses random tarot cards and making request to the AI API to generate prediction based on the type, user's prompt and tarot cards.

## Stack

**Frontend** *(TypeScript)*:
- React
- TailwindCSS

**Backend** *(Python)*:
- fastAPI
- Pydantic
- SQLalcemy
- google.genai _(Google generative AI API)_

As database to store all the tarot cards we used SQLite.

## Usage

> Requires Python 3.12+, npm or yarn

Clone the repository:
```bash
git clone https://github.com/yeghor/Tarot.git
cd Tarot
```

Create `.env` file in `backend` directory and fill it with your **Gemini API Key**:
```.env
GOOGLE_API_KEY="YOUR-SUPER-SECRET-API-KEY"
```

Move to backend directory:
```bash
cd backend
```

Create and activate python's virtual environment _(optional)_:
```bash
python -m venv venv
venv/Scripts/activate.bat # Windows
source my_venv/bin/activate # MacOS/Linux
```

Install the dependencies:
```bash
pip install -r requirements.txt
cd ..

cd frontend
npm i # or yarn install
cd ..
```

Run backend and frontend:
```bash
cd backend
uvicorn main:app --reload

cd ..

cd frontend
npm run dev # yarn dev
```

**Access**:
- [Backend](http://localhost:8000/docs) _(Swagger autogenerated API documentation)_
- [Frontend](http://localhost:5173)
